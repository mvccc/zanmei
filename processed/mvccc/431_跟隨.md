# 跟隨

## (1)
我願意跟隨主 奔走平坦道路
或遇花木茂盛 清水常流之處
既有救主引導 我心煥然跟隨主
一路走到天上 我必跟隨主

## (1)
跟隨跟隨 我願跟隨耶穌
不論往何地方 我要跟隨主
跟隨跟隨 我願跟隨耶穌
只望耶穌引導 我必跟隨主

## (2)
我願意跟隨主 奔走危險道路
或遇狂風暴雨 黑雲阻我前途
既有救主引導 我心永遠不受驚
總不懼怕危險 因有主同行

我願意跟隨主 不論高山平地
我靈常願與主 親近永不分離
他必安然領我 跟隨他的腳蹤行
我願與主同行 一直到天庭
